# Python Projects Collection

A hands-on collection of beginner-friendly Python mini-projects designed to strengthen core programming skills through practical, interactive builds.

This repository includes small applications such as a calculator, number guessing game, password generator, to-do list, dice roller, file handling app, and more. Each project focuses on fundamental Python concepts like variables, loops, conditionals, functions, lists, dictionaries, randomization, file handling, and basic web/UI development.


## Overview

These projects are ideal for learning Python step by step while building small, interactive apps.

## Getting Started

### Prerequisites

- Python 3.8 or newer
- pip
- Optional: Streamlit for web-based projects

### Install Streamlit (if needed)

```bash
pip install streamlit
```

## How to Run the Projects

### Console-based projects

Navigate to the project folder and run the main Python file:

```bash
cd "Calculator"
python main.py
```

### Streamlit-based projects

Some projects use a web interface and can be run with Streamlit.

Example:

```bash
cd "Dice Roller"
streamlit run dice_roller_app.py
```

Other Streamlit apps include:

```bash
cd "file handening"
streamlit run app.py
```

```bash
cd "To-Do List"
streamlit run todo_list_app.py
```
